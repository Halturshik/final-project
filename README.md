Проект "Планировщик задач"

Работает с задачами, создаваемыми пользователями.
Умеет добавлять, редактировать, удалять, отмечать выполненными, фильтровать через поле для поиска задач.


Реализованные задания "со звездочкой":
- Определение порта извне. Слушает порт переменной окружения TODO_PORT, если есть
- Определение пути к файлу базы данных через переменную окружения TODO_DBFILE
- Правила повторения "w" и "m" в nextdate
- Фильтрация задач через поле для поиска
- Создание docker-образа

Нереализованные задания "со звездочкой":
- Аутентификация через JWT


Адрес для открытия в браузере: http://localhost:7540


Для запуска тестов следует использовать следующие параметры:
var Port = 7540
var DBFile = "../scheduler.db"
var FullNextDate = true
var Search = true
var Token = ``


Команды для сборки и запуска docker-образа:
docker build -t final-project-image <путь к проекту>
docker run -it --rm -p 7540:7540 -v <путь к проекту>:/db final-project-image
